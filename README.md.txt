
<h1 align="center">Cardápio Digital</h1>

## :memo: Descrição
Essa aplicação foi desenvolvido durante o video [Construindo aplicação Fullstack do ZERO](https://www.youtube.com/@kipperdev) utilizando **Java Spring e Spring MVC** para criação do servidor, **Spring Data JPA** para manipulação e persistência de dados, **Lombok** para geração de boilerplates e MySQLDriver para realizar a conexão com banco de dados MySQL.

## :books: Funcionalidades
 * Cadastrar novos pratos no cardapio
 * Listar itens do Cardapio
 * modificar itens do Cardapio
 *  Deletar itens do Cardapio

## :wrench: Tecnologias utilizadas
* Java
* Spring;
*  MySQL;

## :rocket: Rodando o projeto
Para rodar o repositório é necessário clonar o mesmo:
```
# Clone o repositório
$ git clone https://github.com/JuliaValerio/backend-cardapio-digital

# Acesse-o
$ cd backend-cardapio-digital
```
